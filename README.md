# 8w-qqbot-Mai
## 这是一个基于cqhttp框架搭建的使用python开发的qq机器人，现实现功能如下：
* help 菜单
* rand 输出随机数
* beauty 从指定api读取图片并发送
* picture 读取本地图片并发送
* trans 文本翻译
* abs 阴阳带师（划掉）文字到图片的转换
* ban/unban 禁止某人和bot互动
* dog 舔狗日记
* anime 搜索番剧的url
* pa 让@的人爪巴

## 如何开始
* 移步此处https://github.com/Mrs4s/go-cqhttp 安装并运行exe
* 补充完整python代码并运行
* **开始愉快的bot之旅吧**
